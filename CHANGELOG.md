## [1.0.0] - 2025-11-10

### Added
- Initial release of Recall Knowledge module
- Enhanced rules engine with PF2e-inspired patterns
- Knowledge management system with detailed tracking
- Real-time socket communication for multiplayer sessions
- Comprehensive API for external module integration
- Support for custom rule elements and modifiers
- Multiple chat output styles (simple, detailed, whisper)
- Configurable knowledge check automation
- Debug mode with detailed logging
- Full TypeScript implementation with modern development tools
- Localization support (English included)
- Custom CSS styling with theme compatibility
- Token HUD integration for quick knowledge checks
- Maximum attempt limiting per creature
- Rule element priority system
- Predicate-based rule evaluation
- Mobile-responsive interface design
- Accessibility improvements with high contrast support
- Comprehensive documentation and examples

### Technical Details
- Built with TypeScript 5.3+
- Uses Vite for fast builds and development
- ESLint and Prettier for code quality
- Foundry VTT v13+ compatibility
- Modern ES2022+ JavaScript features
- Comprehensive type definitions for Foundry VTT
- Modular architecture with clean separation of concerns
- Socket-based real-time communication
- Event-driven hook system
- Extensible rule processing engine
